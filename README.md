# Factorial Calculator

This project demonstrates two different Python implementations for calculating the factorial of a number. The first version uses Python's built-in `math.factorial()` function to compute the result efficiently, while the second version manually calculates the factorial using a `while` loop. These two approaches allow beginners to compare the convenience of built-in functions with the logic behind implementing the algorithm themselves.

## Features

* Includes two different implementations of a factorial calculator.
* Version 1 uses the `math.factorial()` function from Python's `math` module.
* Version 2 manually computes the factorial using a `while` loop.
* Accepts user input for a positive integer.
* Displays the factorial of the entered number.

## Project Structure

* `factorial_math.py` – Calculates the factorial using the built-in `math.factorial()` function.
* `factorial_manual.py` – Calculates the factorial manually using a `while` loop.

## How to Run

1. Make sure Python 3 is installed.
2. Run either version:

   ```bash
   python factorial_math.py
   ```

   or

   ```bash
   python factorial_manual.py
   ```
3. Enter a positive integer when prompted.
